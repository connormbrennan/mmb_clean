## Notes

This task owns the real-rigidity table group formerly produced by the combined paper-table task.
Table 8 uses the four current channel indicators plus open economy, with all pairwise interaction candidates among those five variables in the full-sample table. The estimated-model appendix table uses main effects only because the all-pairwise specification has sparse model-level support in that subsample. Cells with fewer than three supporting models are blanked, and support and dropped-term diagnostics are written to CSV.
The citation-weighted tables use the `citation_weight` column from the constructed regression dataset and renormalize it to mean one within each regression sample.
Use `code/main.ipynb` only for interactive inspection and temporary specification previews. Production artifacts are generated by `code/main.py` through `make`.
