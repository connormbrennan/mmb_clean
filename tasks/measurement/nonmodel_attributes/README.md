# Nonmodel Attributes

## Purpose

Estimate how nonmodel attributes such as author affiliation, model size, and model vintage relate to the MMB outcome and timing measures.

## Inputs

- `input/MMB_reg_format.dta`: symlink to `tasks/data/build_mmb_analysis_dataset/output/MMB_reg_format.dta`
- `config/params.yaml`: outcome list, sample restriction, labels, and table specification

## Outputs

- `output/nonmodel_attributes_full_sample.txt`
- `output/nonmodel_attributes_estimated_models.txt`
- `output/table9_nonmodel_attributes_full_sample.tex`
- `output/table9_nonmodel_attributes_full_sample.csv`
- `output/table9_nonmodel_attributes_estimated_models.tex`
- `output/table9_nonmodel_attributes_estimated_models.csv`
- `output/table9_nonmodel_attributes_full_sample_citation_weighted.tex`
- `output/table9_nonmodel_attributes_full_sample_citation_weighted.csv`
- `output/table9_nonmodel_attributes_estimated_models_citation_weighted.tex`
- `output/table9_nonmodel_attributes_estimated_models_citation_weighted.csv`
- `output/manifest.csv`

## Notes

This task owns the nonmodel-attribute table group formerly produced by the combined paper-table task.
Table 9 uses central-bank authorship, log number of equations, middle/late vintage indicators, and the central-bank-authors by late-vintage interaction, with early vintage as the omitted vintage group.
The citation-weighted tables use the `citation_weight` column from the constructed regression dataset and renormalize it to mean one within each regression sample.
Use `code/main.ipynb` only for interactive inspection and temporary specification previews. Production artifacts are generated by `code/main.py` through `make`.
