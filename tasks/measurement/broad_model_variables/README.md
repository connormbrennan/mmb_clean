## Notes

This task owns the combined broad-variable table group formerly produced by the combined paper-table task.
The full-sample Table 10 combines nominal-rigidity variables, the current real-channel variables, open economy, all pairwise interaction candidates among the five real-channel variables, and nonmodel attributes.
The estimated-model appendix table uses a simpler main-effects broad specification because the all-pairwise real-channel specification is too sparse in the estimated-model subsample.
The current real-channel variables are constrained household demand, firm balance sheet, bank lending, labor friction, and open economy.
The citation-weighted tables use the `citation_weight` column from the constructed regression dataset and renormalize it to mean one within each regression sample.
Support CSVs report unique-model support for each displayed term and outcome. The identification-diagnostics CSV reports original, kept, and dropped design columns for each regression.
Use `code/main.ipynb` only for interactive inspection and temporary specification previews. Production artifacts are generated by `code/main.py` through `make`.
