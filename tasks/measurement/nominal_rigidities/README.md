# Nominal Rigidities

## Purpose

Estimate how nominal-rigidity variables relate to the MMB outcome and timing measures.

## Inputs

- `input/MMB_reg_format.dta`: symlink to `tasks/data/build_mmb_analysis_dataset/output/MMB_reg_format.dta`
- `config/params.yaml`: outcome list, sample restriction, labels, and table specification

## Outputs

- `output/nominal_rigidities_full_sample.txt`
- `output/nominal_rigidities_estimated_models.txt`
- `output/table7_nominal_rigidities_full_sample.tex`
- `output/table7_nominal_rigidities_full_sample.csv`
- `output/table7_nominal_rigidities_estimated_models.tex`
- `output/table7_nominal_rigidities_estimated_models.csv`
- `output/table7_nominal_rigidities_full_sample_citation_weighted.tex`
- `output/table7_nominal_rigidities_full_sample_citation_weighted.csv`
- `output/table7_nominal_rigidities_estimated_models_citation_weighted.tex`
- `output/table7_nominal_rigidities_estimated_models_citation_weighted.csv`
- `output/manifest.csv`

## Notes

This task owns the nominal-rigidity table group formerly produced by the combined paper-table task.
The citation-weighted tables use the `citation_weight` column from the constructed regression dataset and renormalize it to mean one within each regression sample.
Use `code/main.ipynb` only for interactive inspection and temporary specification previews. Production artifacts are generated by `code/main.py` through `make`.
