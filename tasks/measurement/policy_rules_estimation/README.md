# Policy Rules Estimation

## Purpose

Estimate how policy-rule indicators and the estimated-model indicator relate to the MMB outcome and timing measures.

## Inputs

- `input/MMB_reg_format.dta`: symlink to `tasks/data/build_mmb_analysis_dataset/output/MMB_reg_format.dta`
- `config/params.yaml`: outcome list, sample restriction, labels, and table specification

## Outputs

- `output/policy_rules_estimation_full_sample.txt`
- `output/policy_rules_estimation_estimated_models.txt`
- `output/table6_baseline_reg_full_sample.tex`
- `output/table6_baseline_reg_full_sample.csv`
- `output/table6_baseline_reg_estimated_models.tex`
- `output/table6_baseline_reg_estimated_models.csv`
- `output/table6_baseline_reg_full_sample_citation_weighted.tex`
- `output/table6_baseline_reg_full_sample_citation_weighted.csv`
- `output/table6_baseline_reg_estimated_models_citation_weighted.tex`
- `output/table6_baseline_reg_estimated_models_citation_weighted.csv`
- `output/manifest.csv`

## Notes

This task owns the policy-rule and estimation-control table group formerly produced by the combined paper-table task.
The citation-weighted tables use the `citation_weight` column from the constructed regression dataset and renormalize it to mean one within each regression sample.
`code/main.ipynb` is exploratory only; production artifacts are regenerated by `code/main.py` through the Makefile.
